最近一直在琢磨一件事，越想越觉得离谱。

以前我们理解软件，特别简单：代码 + 数据。代码是固定的——你用微信，我也用微信，咱俩装的那个 App 底层代码一模一样。区别在数据：你的聊天记录、你的朋友圈、你的支付记录，这些是你独有的。

整个软件行业几十年来都是这个模式。开发者写代码，打包发布，所有人用同一份。用户往里灌自己的数据。代码是静态的，数据是动态的。清清楚楚，泾渭分明。

然后龙虾来了，把这条线给搅浑了。

我在龙虾上写了一堆 skill。写知乎小说的 skill，回答知识类问题的 skill，维护网站的 skill，查数据的 skill。这些 skill 就是代码——它是指令，告诉 AI 该干什么、怎么干。

但这些代码跟传统软件的代码完全不是一回事。

传统代码是所有人共享一份的。我的 skill？只有我有。因为它是根据我自己的知识、我的偏好、我的场景写的。别人写同样功能的 skill，内容肯定和我的不一样。

更关键的是：我的 skill 会自己变。

前一篇文章写过了，我让写小说的 skill 每天复盘自己的数据，把经验回写进去。几周下来，这个 skill 已经被改得面目全非了。它现在的样子跟我最初写的时候差了十万八千里。这些变化不是我做的——是它自己根据运行产生的反馈数据，一点一点改出来的。

代码变成了流动的东西。它跟着数据一起流动，跟着反馈一起进化。你分不清楚哪部分是"代码"哪部分是"数据"了，它们混在一起了。

---

这件事想深了挺让人头大的。

以前做软件工程师，核心技能是什么？写代码。写好代码。写高质量的、可维护的、高性能的代码。

现在呢？写 skill。而 skill 这东西，技术含量真没那么高。你看我那个写知乎小说的 skill，本质上就是一份几页的 Markdown 文档，里面写了一堆规则和约束。没有算法，没有数据结构，没有设计模式。就是一堆自然语言描述的规则。

这意味着什么？意味着写 skill 的门槛比写传统代码低太多了。一个不会编程的设计师、一个产品经理、一个做内容的自媒体人，都能写出自己的 skill。你只需要知道你想让 AI 干什么，以及干的时候有哪些约束——这些都是业务知识和经验判断，不是编程技术。

我前两天干了一件事挺能说明问题的。有个朋友晚上在微信上跟我说，他想做一个 MCP server 给各个 client 提供一个检索 skill 的功能。他说他折腾这事已经折腾好久了——因为他在想怎么封装接口、怎么处理认证、怎么做缓存。

我听完以后直接让小虾写了一个在我的 agentskill.work 网站上检索 skill 的 skill，从动手到开源，七分钟。

七分钟。

他用传统程序员的思路去做这件事，搭 server、写 API、处理边界条件。我用 skill 的思路去做这件事，告诉 AI"你去这个网站搜一下这个关键词然后把结果整理出来"。完事。

我当时有一种很强烈的感觉：程序员们互相卷开源，为了证明"我代码写得比你好"，自尊心驱动着一个比一个卷——结果最后把自己的路给堵死了。你开源了，我比你写得更好我也开源了，他比我们俩都好他还开源了。到最后所有人都白干了，因为好东西全是免费的。

共产主义。

---

说到这儿就不得不聊一个尴尬的话题：skill 赚钱吗？

直说吧：不赚。

skill 天生就是开源的。它就是一份文本文件，你卖出去一份，别人就拿到了全部内容。你没法像 SaaS 那样持续收费，因为它不需要你的服务器来运行——跑在用户自己的龙虾上就行了。

昨天直播的时候大家聊到这个话题，孙老师半开玩笑说"那我们的商业模式还剩什么？打赏？"直播间笑成一片，但笑完细想确实有点发毛。

我们做了一个假设：如果 skill 能变得很强，强到用户用 skill + 自己的 AI 助理就能完成以前需要付费产品才能干的事——那中间层的付费产品就尴尬了。你卖的是工具，但现在工具变成了一段可以免费传播的文本。

当然也有人在想别的路。直播里提到一个叫张浩洋的哥们，之前做和平精英策划的，现在搞了一个 skill 共享平台。思路是：skill 不下载到本地，你通过平台调用，调一次收一次钱。加了信用体系、虚拟货币、议会机制——毕竟人家搞游戏出身的，这套运营体系玩得溜。据说最近被各种投资人找，各种播客采访。

能不能跑通我不知道。但至少有人在认真想这个问题了。

---

回到"代码变成流动资产"这件事。

我觉得这个变化的深层含义是：软件不再是一个"产品"了，它变成了一种"能力"。

以前你的能力体现在你会什么技术、写什么代码。现在你的能力体现在你有什么 skill、这些 skill 能帮你做什么。而 skill 是跟着你走的——你换平台、换 AI 模型、换设备，skill 带着就走了。它是你的数字资产，就像你的知识和经验一样。

agentskill.work 这个网站我全氛围编程搭建的，一行代码没手写过。每天定期从 ClaudeHub 上捞热门 skill 聚合展示。这种网站以前我得花几天时间搭，现在就是跟龙虾聊两个小时就出来了。代码不值钱了。值钱的是你的判断力：你知道该聚合什么、怎么展示、用户需要什么。

这可能就是 AI 时代最大的转变：从"写代码"到"做决策"。代码本身在流动、在贬值、在被 AI 批量生产。稀缺的是知道该写什么代码、该设计什么循环、该建什么闭环的那个判断力。

所有循环都是可以在人为设好以后让 AI 自动迭代的。关键在于你得知道该设什么循环。这个，目前还只有人能干。

目前。
